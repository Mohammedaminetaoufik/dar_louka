## 7. Technical SEO

### robots.txt
- Allows all search engines
- Disallows: /api/, /admin/, /.next/, /node_modules/
- Includes sitemaps for English and French versions
- Specific crawl delays for different bots

### Sitemap Configuration
- Multilingual sitemap with hreflang alternates
- Priority levels:
  - Home: 1.0
  - Rooms: 0.95
  - About: 0.9
  - Contact: 0.8
  - Events: 0.8
  - Gallery: 0.7
- Change frequency strategy:
  - Daily: Home page (dynamic content)
  - Weekly: Rooms, Events (availability changes)
  - Monthly: About, Gallery, Contact (static content)

### Canonical Tags
- Set to: https://darlouka.com
- Prevents duplicate content issues

### Meta Tags
- Geo-positioning tags for local SEO
- ICBM coordinates for geographic mapping
- Language and region specifications
